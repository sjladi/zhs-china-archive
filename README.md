# zhs-china-archive

本仓库用于归档若干独立的 HTML 页面。这些页面来自互联网公开内容，以静态 HTML 文件的形式存放，供查阅与研究使用。

## 项目简介

这是一个以归档为目的的仓库，主要收录各类独立网页的 HTML 副本。仓库不针对特定域名或网站，所收录的页面均保持原始内容与结构，仅作存档用途。

## 目录说明

```
zhs-china-archive/
├── pages/          # 存放已归档的 HTML 页面文件
├── assets/         # 页面引用的样式、脚本、图片等资源文件
├── index.html      # 仓库首页（可选，用于展示归档列表）
└── README.md       # 本文件
```

- **pages/**：每个独立页面以 `.html` 文件形式存放，文件名通常与页面标题或内容相关。
- **assets/**：页面所需的额外资源，按页面或类型组织。

## 页面归档说明

- 归档内容为静态 HTML 文件，未修改原始内容。
- 部分页面可能依赖外部资源（如图片、字体、样式表），仓库会尽量保留可内嵌的资源，无法内嵌的将保留原始链接。
- 页面时间戳以归档时的日期为准，如需查看原始发布时间，请参考页面内标注。

## 维护说明

- 本仓库不定期更新，新增归档页面会通过提交记录体现。
- 欢迎通过 Issue 提出归档请求或反馈问题，但无法保证所有请求均被处理。
- 如需使用仓库内容，请遵守相关法律法规及原始页面版权声明。
- 仓库维护者不对页面内容准确性负责，亦不保证所有链接与资源永久可用。
